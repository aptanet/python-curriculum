#Introdução:
Este projeto ensina as crianças como utilizar o loop `while` e números aleatórios para criar diversas versões de um jogo de adivinhações.

#Recursos
Para este projeto, o Python deverá estar instalado. É recomendado que a versão 3.2 do Python esteja instalada.

As crianças também poderão fazer uso dos materiais que acompanham estes desafios. Arquivos incluídos na pasta de "Recursos do Projeto" (que fica localizada no link "Download de Materiais do Projeto"):

+ PortaDaFortuna.py

Tenha certeza de que cada criança tem acesso de "read" e "write" (leitura e escrita) nas cópias desses arquivos em seus computadores.

Você pode achar os arquivos completos dos desafios deste projeto ao clicar no link de "Download de Materiais do Projeto", que contém:

+ KeepingScore.py
+ TwentyOne.py

#Objetivos de Aprendizado
+ Usar loops `while` em um programa;
+ Usar o módulo `random`;
+ Adicionar comentários a um programa;
+ Tipo Booleano de dados 

#Desafios
+ Contando os pontos - usando uma variável para contar a pontuação do jogador;
+ Consertando a entrada - utilizar uma entrada de dados mais robusta
+ Perdendo o jogo - mudando a lógica do jogo;
+ Vinte-e-um - usando o loop `while` e números aleatórios para criar um jogo ao estilo blackjack;
